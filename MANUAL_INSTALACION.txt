PASO 1: Iniciar XAMPP
Ejecutar XAMPP y encender los servicios de Apache y MySQL.

PASO 2: Base de datos
Abrir la carpeta db.
Dentro se encuentra el script de la base de datos, ejecutarlo en MySQL para crear la base de datos.

PASO 3: Backend (Node.js)
Abrir la carpeta "Base de datos JS" y luego ingresar a "api_basic".

Dependencias:
bcrypt, bcryptjs, cors, dotenv, express, jsonwebtoken, morgan, mysql2, nodemon.

Instalar dependencias:
npm install

Si alguna librería presenta errores, reinstalarla ejecutando primero el uninstall y luego el install:

npm uninstall bcrypt - npm install bcrypt
npm uninstall bcryptjs - npm install bcryptjs
npm uninstall cors - npm install cors
npm uninstall dotenv - npm install dotenv
npm uninstall express - npm install express
npm uninstall jsonwebtoken - npm install jsonwebtoken
npm uninstall morgan - npm install morgan
npm uninstall mysql2 - npm install mysql2
npm uninstall nodemon - npm install nodemon

Para ejecutar el backend:
npm run dev

PASO 4: Frontend (Flutter con Dart)
Abrir la carpeta "login_registrorol_home" y abrir una terminal dentro del proyecto.

Ejecutar:
flutter clean
flutter pub get

Luego ejecutar el proyecto:
flutter run
